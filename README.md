### node-blog
一个用node写的博客

### 功能介绍
使用node搭建的多人博客，使用express框架，ejs模板引擎，mongoDB数据库，包含注册、登陆、登出、发布文章、编辑文章、删除文章、添加留言、删除留言、增加日志等等功能

### 技术细节
利用mvc架构，实现的一个博客。

### 使用技能：
node ejs mongoDB

### 使用
`git clone`

`cd node-blog`

`npm i`

`node index.js`
